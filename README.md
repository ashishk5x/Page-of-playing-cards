# 🃏 Playing Cards Layout

A simple and responsive **Playing Cards Layout** built using **HTML5 and CSS3**. This project recreates classic playing cards (Ace through Five) using only HTML and CSS, demonstrating the power of Flexbox, transforms, spacing, hover effects, and responsive layouts without any images or external libraries.

## ✨ Features

* Classic playing card design
* Different card suits (♠ ♣ ♥ ♦)
* Color-coded red and black cards
* Hover animation with scaling effect
* Box-shadow depth effect on hover
* Responsive Flexbox layout
* Rotated card values for authentic card appearance
* Pure HTML and CSS implementation

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox

## 🚀 How to Run

1. Clone the repository

```bash
git clone <repo-link>
```

2. Navigate to the project folder

```bash
cd playing-cards-layout
```

3. Open `index.html` in your browser

No additional setup is required.
